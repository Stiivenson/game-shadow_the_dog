# game-shadow_the_dog
2D scrolling game in canvas.

## Development

Project uses Vite as a bundler.

### Initialize project

Clone repo:
```
git clone https://github.com/Stiivenson/game-shadow_the_dog
```
Go to project folder:
```
cd game-shadow_the_dog
```
Install deps:
```
npm ci
```  
Start dev server:
```
npm run dev
```

## Production

Build project bundle:
```
npm run build
```
